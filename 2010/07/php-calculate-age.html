PHPで年齢算出

PHP

方法1

<?php
    //生年月日を指定
    $birth = "19990401";
    echo floor((date("Ymd") - $birth) / 10000);
?>

方法2

<?php
    $birth = "1990-06-06";
    $now = date("Y-m-d");

    list($y1, $m1, $d1) = explode("-", $birth);
    list($y2, $m2, $d2) = explode("-", $now);

    $age = ($y2 - $y1);

    // 誕生日を過ぎていない場合は1を引く
    if(($m1 . $d1) > ($m2 . $d2)) {
        $age--;
    }

    echo $age;
?>

Unixタイムスタンプ系関数を使ってやる方法

<?php
    $birth = "1981-05-14";
    echo floor((time() - strtotime($birth)) / (60 * 60 * 24 * 365));
?>
