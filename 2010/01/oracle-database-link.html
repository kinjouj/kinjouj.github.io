Oracle Database Link

Database Oracle

別サーバー上のOracleデータベースを操作する際に利用する機能

TNSを設定

sub01 = (DESCRIPTION =
    (ADDRESS =
        (PROTOCOL =TCP)
        (HOST = subora.example.com)
        (PORT = 1521)
    )
    (CONNECT_DATA =
        (SID = subora)
    )
)

データベースリンクを作成

CREATE DATABASE LINK DBL1
    CONNECT TO hoge
    IDENTIFIED BY hoge
    USING "tns:sub01";

使用する際には

SELECT ID,NAME FROM SAMPLE@DBL1;

のように@データベースリンク名を指定する事で可能らしい
