cat - な件

Linux

このブログの記事書いてる時に

<>だけをエスケープするコマンドが必要だ

— kinjouj (@kinjou_j) September 21, 2013

って思ってしまった訳で、でこのブログ以外で書く事まず無いんだけど.bashrc辺りにfunction定義しちゃえば良いんじゃないかって

そこで

cat src/main/resources/struts.xml | escape_kakko

的な感じで出来るようにpipe?を利用する.bashrcなfunction定義でどうするんでしたっけという事でググった所、cat -を利用する事で出来る模様

function escape_kakko {
    cat - | sed -e "s/</</g" | sed -e "s/>/>/g"
}

余談、<とか>ってAngle Bracketsって呼ばれるんですかね。ググってる途中でそういうっぽいのを見ただけだけど知らなかった
