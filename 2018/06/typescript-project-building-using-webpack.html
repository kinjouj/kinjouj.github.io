webpackでTypeScriptプロジェクトをビルド

2018-06-17T00:00:00+09:00 JavaScript webpack

前回のちょっとした続き

※前回はこちら

まぁTypeScriptをwebpackで処理するんならまぁts-loader使えば良いんでしょ的なチャラい事言ってやってなかったので一応やってみたら普通に出来たのでおｋ

— JK (@kinjou__j) 2018年6月16日

っていうのをやってみた

webpack.config.js

module.exports = {
  mode: "development",
  entry: "./src/index.tsx",
  output: {
    filename: "index.js",
    path: __dirname + "/build"
  },
  resolve: {
    extensions: [".ts", ".tsx", ".json", ".js"]
  },
  devServer: {
    contentBase: "build"
  },
  module: {
    rules: [
      {
        test: /\.tsx$/,
        use: {
          loader: "ts-loader",
          options: {
            transpileOnly: true
          }
        }
      },
      {
        test: /\.js$/,
        loader: "babel-loader",
        exclude: /node_modules/
      }
    ]
  }
};

んまぁts-loader使えばいいだけみたいよねって事で終わり
